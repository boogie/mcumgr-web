Web Bluetooth Device Manager

This tool is compatible with desktops (or laptops) with the latest Chrome and Edge browsers (other Chromium-based browsers may also work), and working Bluetooth connection (most laptops have them these days). You can also try it from Chrome on Android, or Bluefy on iOS/iPadOS.

Troubleshooting tips:
  • • Make sure the device is powered on and in range
  • • Check if the device is already connected to another app or browser tab
  • • Try forgetting the device in your OS Bluetooth settings and reconnect
  • • Power cycle the device (turn it off and on)
  • • Refresh this page and try again
